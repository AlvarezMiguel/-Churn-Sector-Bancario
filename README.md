# Análisis Predictivo de Abandono de Clientes (Bank Churn)
Este proyecto desarrolla un sistema de clasificación para predecir la probabilidad de que un cliente cancele su cuenta bancaria. Se implementó un flujo completo de análisis, con enfasisi en el tratamiento de datos desbalanceados y la creación de un modelo de ensamble.

## Desafíos Técnicos
**Desbalance de Clientes: La base original presentaba un 80% de permanencia vs. un 20% de abandono. Se solucionó utilizando SMOTE-NC para equilibrar las clases y evitar sesgos en el entrenamiento (p. 3)**
